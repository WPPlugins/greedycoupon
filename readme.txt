GreedyCoupon is an free wordpress coupon plugin which help you to display coupon in site and allow you to make coupon page in any wordpress website.

== Description ==

Start with GreedyCoupon by making coupon pages and use hidden coupon buttons to get more sales from affiliate.We make it to help those web masters who need to add own coupon pages or want to make a coupon website without using any coupon theme.
GreedyCoupon Features
-> Built in Custom Coupon Post/page design,No Need to buy theme
-> Add multiple coupons in one page
-> Show coupons post with slider
-> SEO compatable coupon pages
Separate affiliate links on each post.

== Installation ==

1. Upload `greedy-coupon.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now you can use GreedyCoupon

== Frequently asked questions ==

* How to use GreedyCoupon coupon slider ?
Ans- GreedyCoupon allow you to use coupons slider anywhere in the website using shortcode.You can use "[gd_coupon_carousel]" in post/pages or even in any plugin/theme code. This will work as it is.
